# algoritmos-ordenacao
Exercício Bubble Sort e Selection Sort com análise de desempenho

Conclusão do código: 

Após executar os testes com vetores de tamanho 100, 1000 e 10000,
foi possível observar que ambos os algoritmos possuem complexidade O(n²),
resultando em crescimento significativo do tempo de execução conforme o tamanho aumenta.

O Selection Sort apresentou melhor desempenho geral, pois realiza menos movimentações
comparado ao Bubble Sort, mesmo tendo número semelhante de comparações.

O Bubble Sort realiza muitas trocas, o que aumenta seu tempo de execução,
principalmente em vetores grandes.

Portanto, o Selection Sort é mais eficiente que o Bubble Sort neste cenário.
